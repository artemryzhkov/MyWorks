Программа отражает коэффициент отношения минимального (условного)
набора продуктов питания к среднедушевым доходам по Федеральным Округам РФ
за 2013-2016 гг.

Также демонстрируется гистограмма данных коэффициентов, диаграмма
средних доходов населения по ФО РФ за 2016 год, диаграмма стоимости 
минимального набора продуктов питания по ФО РФ за 2016 год.

Запуск файла "coefficient of the ratio of prices to incomes.py".

Представлены соответствующие таблицы Excel с официального сайта Росстата,
в частности таблица стоимости минимального набора продуктов питания по 
субъектам РФ за 2013-2016 гг. и таблица среднедушевых доходов по субъектам РФ
за 2013-2016 гг., которые были использованы при вычислениях.
